FullTextType(easy_analyzer)

FullTextType FullTextType is an implementation of FieldType tuned for “full text search”. Full text fields are associated with an Analyzer, which is used to tokenize and normalize the text so that it can be searched for individual words. For an exact-match, single value field type using character data, see StringType.
